# Ilztalbahn Fahrgastzählung – Version 2.0

## Installation
1. ZIP entpacken.
2. Alle Dateien und den Ordner `icons` in das bestehende GitHub-Repository hochladen und vorhandene Dateien überschreiben.
3. `Commit changes` anklicken.
4. 1–2 Minuten warten und die GitHub-Pages-App neu laden.

## Funktionen
- Logo und PWA-App-Symbole
- Offline-Erfassung
- automatische Google-Sheets-Synchronisation
- automatische Fahrtrichtung
- Dashboard
- Deutschlandticket, Bayernticket, Guti Ticket, Fahrrad und Nicht angegeben
- OD-Matrix
- Streckenbelegung
- Auswertung nach Zugnummer
- CSV-Export
- Übernahme lokaler Daten aus älteren Versionen

## iPhone-Aktualisierung
Bei alter Anzeige die App vom Home-Bildschirm entfernen, die GitHub-Pages-Adresse in Safari neu öffnen und wieder zum Home-Bildschirm hinzufügen.
