Ilztalbahn PWA - Installation

Änderung:
- Fahrscheinart ist optional.
- Wird nichts gewählt, wird "Nicht angegeben" gespeichert und exportiert.

Dateien:
- index.html
- style.css
- app.js
- manifest.json
- service-worker.js

Für iPhone/iPad als App:
1. Dateien auf Webserver/GitHub Pages/Netlify hochladen.
2. Link in Safari öffnen.
3. Teilen -> Zum Home-Bildschirm.
